2 versions of luma on 1 sd

This is good for having both the features of
the newest Luma and the ability to use NTR CFW
What you'll need:

  • Computer (Doesn't matter what OS you have)
  • 3ds with B9S
  • SD card (You don't need that much free space)

Let's start now!

  1. Download the 2 releases you want from Github
  2. Copy the one you want to boot by default to root of SD (replacing any files)
  3. Copy the other one to sdmc:/luma/payloads
  4. Eject SD card and put into 3ds

If you did all that, then you can boot into another version
of Luma by holding START on boot and choosing the file named "boot".
