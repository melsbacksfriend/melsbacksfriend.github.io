Safer alternative to eMMC upgrade

I'M NOT RESPONSIBLE FOR BANS OR SD CORRUPTIONS!
What you'll need:

  • Unix based OS (Again, using Ubuntu)
  • Hackable Switch
  • Python
  • resize-user.py
  • Hekate
  • SD with capacity >=64GB
  • (if using Ubuntu) Gparted
  • CFW (You can use any cfw except ReiNX, even Hekate but I use atmosphère)
  • python-mbedtls
  • Lockpick RCM

Let's go! (Like Pokémon Let's Go Eevee, My favorite Switch game)

  1. Go into Hekate
  2. Back up both raw GPP and BOOT0/BOOT1
  3. Back up keys using Lockpick RCM
  4. Put SD into computer
  5. Open Terminal
  6. Execute this command without quotes and replacing PATH TO BOOT0, PATH TO BOOT1 and PATH TO RAWNAND.BIN with the paths to those files "cat PATH TO BOOT0 PATH TO BOOT1 PATH TO RAWNAND.BIN > emummc.bin"
  7. Execute this command without quotes "sudo python3 pip install --user python-mbedtls"
  8. Execute this command without quotes and replacing PATH TO RESIZE-USER.PY, PATH TO EMUMMC.BIN and PATH TO PROD.KEYS with the paths to those files and replacing SIZE with partition size in GB "python3 PATH TO RESIZE-USER.PY PATH TO EMUMMC.BIN SIZE PATH TO PROD.KEYS"
  9. Open Gparted
  10. Select your SD card
  11. Right click on the only partition there and select "Resize/Move"
  12. Make the unallocated space after be the size you want (MUST BE >=THE FILE SIZE OF EMUMMC.BIN!)
  13. Right click on the unallocated space and create a partition that fills the unallocated space (Format doesn't matter)
  14. Apply the operations
  15. Open the terminal again
  16. Execute without quotes and replacing /dev/sdxy with the location of the new partition and PATH TO EMUMMC.BIN with the path to that file "sudo dd if=PATH TO EMUMMC.BIN of=/dev/sdxy"
  17. Eject your SD card and put it into your Switch
  18. Go back into Hekate
  19. Choose emummc
  20. Select SD RAW partition 1
  21. Boot into your favorite CFW (not ReiNX)

That was a LOT of steps but now you're done!
