Have big file on fat32

What you'll need:

  • Unix based OS on your computer (I'm using Ubuntu)
  • Hackable Switch
  • fat32 sd card
  • Some way to boot Switch into CFW
  • NXshell (available on HB app store)

Without further ado, let's get started!

  1. Get the file you want
  2. Open the terminal
  3. Execute this command without quotes and replacing PATH TO FILE with the path to the file you're using "split -d -a 1 -b 4294901760 PATH TO FILE 0"
  4. Create a folder on your sd card. THE FOLDER'S NAME SHOULD END WITH THE EXTENSION OF THE FILE YOU STARTED WITH (example: I start with ssbu.nsp and name the folder ssbu_split.nsp)
  5. Copy all of the files that appeared after splitting the original file into the folder you created
  6. Eject sd card and put into Switch
  7. Boot Switch into CFW using whatever method you prefer
  8. Go into hbmenu and choose NXshell
  9. Enable developer mode in NXshell's settings
  10. Select folder and choose "Set archive bit".

Now your Switch should treat the folder as a file made up of the folder's contents merged.
